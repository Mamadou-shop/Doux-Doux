#Doux-Doux - Boutique Multi-Catégories

Bienvenue sur le dépôt de **Doux-Doux**, une application web moderne conçue pour une expérience de shopping fluide et rapide.

## 📦 Nos Rayons
- **High-Tech** : Derniers modèles d'iPhones et smartphones.
- **Mode & Vêtements** : Sélection de prêt-à-porter tendance.
- **Chaussures** : Baskets et souliers pour tous les styles.
- **Accessoires** : Compléments indispensables pour vos appareils et votre look.

## ✨ Fonctionnalités Clés
- **Filtrage Dynamique** : Navigation instantanée entre les catégories sans rechargement de page.
- **Affichage Intelligent** : Chargement automatique des produits depuis une base de données JSON.
- **Tunnel d'Achat Sécurisé** : Modale de résumé de commande affichant dynamiquement le produit choisi et les options de paiement (Wave, Orange Money).
- **Design Responsive** : Optimisé pour une utilisation sur mobile et ordinateur.

## 🛠️ Stack Technique
- **Frontend** : HTML5 pur et CSS3 personnalisé pour un design épuré.
- **Logique** : JavaScript Vanilla (ES6) pour la gestion du panier et des événements.
- **Données** : Format JSON pour une gestion simplifiée de l'inventaire.
